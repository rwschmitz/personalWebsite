I'm a pretty big nerd and to demonstrate that... I present to you my VBA project that I wrote to help my World of Warcraft guild more easily recruit players. Yeah. Link to the code on GitHub is at the bottom. Here's a short description, though:



1. Open Microsoft Excel and click the Import Players button.

2. A dialogue box will open prompting you to enter a server name (think of this like a job seeker's city/state location).

3. A dialogue box will open asking how many pages you wish to run. These pages correspond to the number of tables on the website wowprogress.com. In short, the more pages you enter, the more players that will be scanned for the server you entered in step #2.


4. The tool will begin running after you hit ok. Behind the scenes, the tool is pulling each table down in to its own sheet and filtering each sheet with the server name specified in step #2. Once all tables (sheets) are populated and sorted, the tool takes all of the resulting information and combines the information into a tab called, "Combined" and deletes all the individual sheets. If you entered 25 pages, then 25 sheets were temporarily created, filtered and aggregated to the "Combined" tab, then subsequently deleted.
